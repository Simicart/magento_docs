## PayPal Mobile Properties



## Save Payment via PayPal Mobile

```shell
curl -X POST "https://abc.com/simiconnector/rest/v2/paypalmobile" \
  -H "Authorization: Bearer <token>" \
  -d "{
"payment_status":"1",
"invoice_number":"100000",
"proof":"<json object from PayPal>",
}
"
```
